# Virtual OSS control
Virtual OSS control is a GUI for virtual_oss.

## How to build under FreeBSD
<ul>
  <li>qmake</li>
  <li>make all</li>
  <li>make install</li>
</ul>

## Dependencies
<ul>
  <li>QT v5.x</li>
  <li><A HREF="http://www.fftw.org">FFTW3</A> </li>
</ul>

## Privacy policy

Virtual OSS control does not collect any information from its users.
